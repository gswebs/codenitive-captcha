Secure WordPress and WooCommerce with easy Google reCAPTCHA integration.

== Description ==

**Enhance your website’s security** by integrating CAPTCHA verification into essential WordPress and WooCommerce forms. JMB Captcha helps prevent spam, bot abuse, and unauthorized access by adding **Google reCAPTCHA** (v2) to critical forms:

- WordPress Login
- Registration
- Lost Password
- WooCommerce Login, Registration, Checkout

Built for performance and ease of use, JMB Captcha requires no coding and integrates smoothly with your site's design.

== Features ==

- 🔐 Add **Google reCAPTCHA** (v2) to:
  - WordPress Login, Registration, Lost Password
  - WooCommerce Login, Registration, and Checkout
- 🛡️ Prevent spam, bots, and brute-force attacks
- ⚙️ Easy configuration via plugin settings panel
- 🧩 Compatible with most themes and caching plugins
- 🧪 Tested with the latest WordPress and WooCommerce versions

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/jmb-captcha`, or install it via the Plugins screen in WordPress.
2. Activate the plugin through the 'Plugins' menu.
3. Go to **Settings → JMB Captcha** to configure your Google reCAPTCHA site and secret keys.
4. Choose which forms to protect and which reCAPTCHA type to use (v2 Checkbox).
5. Save your settings.

== Frequently Asked Questions ==

= Which reCAPTCHA versions are supported? =  
Google reCAPTCHA v2 (Checkbox).

= Does this plugin work with WooCommerce? =  
Yes! It supports login, registration, and checkout forms in WooCommerce.

= Will it slow down my site? =  
No. The plugin is lightweight and optimized to only load CAPTCHA scripts on pages where needed.

= Is this plugin compatible with caching plugins? =  
Yes, it works with major caching plugins like WP Super Cache, W3 Total Cache, and LiteSpeed.

= Can I customize where CAPTCHA appears? =  
Yes, you can enable or disable CAPTCHA per form via the settings page.

= What if reCAPTCHA doesn't show? =  
Make sure you've entered valid **site key** and **secret key** from the Google reCAPTCHA admin panel, and that the form you’re testing is enabled in settings.
